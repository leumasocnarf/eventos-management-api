## Endpoints

### Create Equipamento
```js
POST api/v1/equipamentos
```

### Read Equipamento
```js
GET api/v1/equipamentos/{id}
```
```js
GET api/v1/equipamentos/
```

### Update Equipamento
```js
PUT api/v1/equipamentos/{id}
```

### Delete Equipamento
```js
DELETE api/v1/equipamento/{id}
```
